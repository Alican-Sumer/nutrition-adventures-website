# Nutrition Ninja

Standalone mini-game draft for the Nutrition Adventures RPG.

Concept:
- Fruit Ninja style slicing loop
- RPG nutrition framing: the player trains by slicing "power foods"
- Penalty targets are "boss snacks" that cost hearts and reset combo flow

Files:
- `NutritionNinja.html`
- `nutrition-ninja.css`
- `nutrition-ninja.js`

Current behavior:
- Standalone draft files only; not currently wired into the main map scene
- Mouse or touch drag creates slash trails
- Good foods add score and combo
- Bad foods remove hearts
- Every 5-hit combo grants a bonus heart
